Git

What is Git?

Git is a version control system, it was originally created for developers to work on open source projects together.

The uses of the Git system

  • Keep track of different version of project
  • Work together by merging different versions
  • Push and Pull the different files
  • Move around the different version of the code

How to use Git

To use Git, the git softwear needs to be installed from websites such as this Git Download open it up, to use the commands in the right place.

Commands for Git

This is a list of the few useful commands to work with Git.

  • pwd - This shows the current working depository
  • touch filename - This command allows for creation of files in the current location (find by using the pwd command)
  • git init - Use this command to create a new git project
  • git status - Use this to check what files have been saved to be commited and which haven't
  • git add - Use this to confirm the new files
