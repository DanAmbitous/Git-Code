Welcome to this website about Git. This part of the website is an over view of Git, as well as helpful links about it.

Git is a verion control system that is used commonly by websites such as github, if you interested in using Git, you can download it here at the its website.

Git was founded by Linus Torvalds, a Finnish-American programmer, who has also notably founded Linux. He wanted a verison-control system unlike any that existed at the time, so he coded one. Some of the aspects of Git is that it supports a non-linear development, distributed development, efficient handling of large projects, cryptographic authentication of history, and a lot more.
